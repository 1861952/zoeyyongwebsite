Grow Your Own Garden Program

In this project, the user first inputs two colors when prompted to.

Next, the draws a spider, a ladybug, and watering can. Wherever the user clicks,

the program draw a flower that contains the colors inputted by the user.

A bee flys back and forth in the background. For and while loops

as well as if-else statements were utilized in this program.


Fruit Ninja Game

This Fruit Ninja Game is a game where the user uses the mouse to click the flying

fruits on the screen to gain points. The goal of the game is to get as many points

as possible before time runs out. This game was created using lists, fuctions, if else statements,

and for loops. Parts of the code are from past projects (i.e. the timer and points counter).

There is also music playing in the background.


Battle Flight Game

In this game, the user uses the arrow keys and space bar to control their plane

while the red enemy planes fly back and forth at the top of the screen.

The user presses the spacebar to fire missiles at the enemy planes. The game ends

when the user moves their plane to close to the enemy planes and collides with them.

A leaderboard pops up after the game is over and shows how many points each user earned.

This game utilizes lists, loops, conditional statements, and functions. The program

also responds to user input and is event-driven.
